Demonstrates

The tabbed pages of the tab control demonstrate different techniques:

  • Add Items   The sample retrieves an array of Process objects representing the processes currently running on the computer. The objects are added to the collection of the controls. The property of the control is used to specify which property of the object is displayed in the control.

  • Bind to DataTable   This tab page has code to fill a with a list of all files in your My Documents folder, and then binds the list box to that .

  • Bind to Array   This tab page has code similar to the Add Items page. Instead of adding the objects one by one to the collection, the whole array of objects is used as the of the control. The and properties of the control are used to control the display and retrieval of the items.

  • ComboBox   This tab page has code to bind a control to a retrieved from SQL Server using the new TableAdapter and typed dataset features. This example expects that it can find the Northwind sample database in SQL Server on the local computer. The page has controls to enable you to specify some of the layout properties that affect the behavior of the control.
